# Folder specific search bar

Omnisearch-powered search bar for one folder or whole vault

## Description

- **Scope:** results are filtered to files whose paths start with `FOLDER`.
- **What to edit:**
  - `FOLDER`: folder path with a trailing slash, e.g. `parent/folder/file`.
  - `MAX`: maximum visible results.
  - `placeholder`: visible search placeholder text.
- **Result display:** note title on the left, created date on the right.
- **Width:** forced to Obsidian reading line width with `--file-line-width`, so wide snippets should not stretch it.
- **Click behavior:** click opens the note; Ctrl/Cmd-click opens in a new leaf.
